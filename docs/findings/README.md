# Founder Findings

These folders are the share links. Each one is written so it can be sent directly to a founder
without asking them to inspect the whole repository.

Every promoted change must clear the adversarially-confirmed to add value bar: same task, same
model or harness cell, baseline compared with tuned wording, and a live receipt.

## How To Use This

Send the folder link when there is a confirmed improvement.

Send a guardrail folder only if the founder wants to see the testing method. Do not pitch a
guardrail as a recommended upstream change.

The full ledger is [Confirmed Improvements](https://github.com/cfregly/claude-agent-harness-opt/blob/main/docs/confirmed-improvements.md).
